# Weather App 🌦️

A simple Weather Application built using React and OpenWeather API.

## 🔗 Live Demo
https://weather-app.vercel.app

## 🚀 Features
- Search weather by city
- Shows temperature and weather condition
- Responsive UI

## 🛠 Tech Stack
- React
- Vite
- Material UI
- OpenWeather API

## 📦 Installation

Clone the repository

git clone https://github.com/deepakcodes18/Weather-app.git

Go to project folder

cd Weather-app

Install dependencies

npm install

Run the app

npm run dev
